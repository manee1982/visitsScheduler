Visits Scheduler

Welcome to technician scheduler visits application, this application schedules and maintain technicians visits to the company clients.

Configurations

Clone the source code from app repository, or download it as zip file.

Upload visitsScheduler/dist/visitsScheduler.war to tomcat web server, or build from source

Import database visits_scheduler.sql to MySQL database server, and create user demo with password demo with permissions.

How to use

  • Click on day in the calendar to schedule visit.
  • fill the dialog by needed data.
  • When successfully added, refresh the page to see scheduled visits in calendar.
  • Click on day that has schedule visit, to show load percentage context.
  • Percentage calculation, according to time rang selected, Ex 12-13 ==> 25% and so on.
