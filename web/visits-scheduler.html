Visits Scheduler

Welcome to technician scheduler visits application, this application schedules and maintain technicians visits to the company clients.

Features:

  • Schedule visits.
  • Manage technician load-time.
  • Display and manage visits in calendar.

Note

This application talks to a Java RESTful (JAX-RS) web service API, in server-side which makes all functionality that needed to save, reserve and manipulate visits data.

This application uses FullCalendar javascript library in client-side.
