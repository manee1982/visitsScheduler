About

Welcome to sceduler visits application, this application schedules and maintain technicians visits to the company clients.

This application uses FullCalendar javascript library in client-side.

This application talks to a Java RESTful (JAX-RS) API, in server-side which makes all functionality that needed to save, reserve and manipulate visits data.
